# 🏘️ Kids Town 2.0 — 開發計劃

## 總結 Design

```
做任務/家課 → 🪙 金幣 + ⭐ 經驗
       ↓
探險 (消耗金幣) → 獲得 🪵木頭 🧱磚頭 🔩鐵
       ↓
收集材料 → 起建築物/升級
       ↓
建築物 buff → 更有效率做任務 + 探險
       ↓
探險特殊事件 → 關鍵材料 → Lv.5 升級
       ↓
解鎖新區域 → 新建築物設計圖 → ...
```
